# Custom-ESP32-WIFI-BLE-Scanner
---
## I created my own handheld network and bluetooth scanning device that simultaniously acts as a deauthentication frame and beacon spam detecter. This handheld device has three main menu options that range from WIFI, BLE, and Attack Detection. The WIFI and BLE option allow the user to scan and display nearby possible connections with the addition of also displaying the RSSI next to each connection.
